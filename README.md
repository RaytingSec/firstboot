First Boot
==============================

## Steps

0. copy configs and firstboot script over
1. update grub config
2. install core packages, configs, etc.
3. gnome configs
4. gnome extensions
5. extra addons and packages as needed

```bash
core.sh
core-{gui,host}.sh
packages.sh
packages-{gui,host}.sh
gnome.sh
gnome-{host,vm}.sh
```
